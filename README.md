# balloon-party
On a basic white canvas, click to place a coloured balloon in the spot. Each one gets a randomized colour, to inflate press the given letter. Press space to pop the most recent one. Up to 26 balloons, one per letter.

## Controls:
- Mouse click : Place a new balloon at given location
- Any letter key : Inflate the balloon with the letter key pressed
- Space : Pop the most recently placed balloon

## Concepts Used:
- Dictories as records
- Random RBG colour generation
- Closures
- List operations
- Event-driven programming
- Data modeling

## To run make sure python is downloaded, then open code on any python editor.
